## How it's built

By hand, on purpose.

Each page is a single self-contained HTML file — markup, styles, and the little JavaScript it needs, all in one place.
```
index.html        English
index-it.html     Italiano
index-es.html     Español
Resources/        artwork, photos, app icons
```
